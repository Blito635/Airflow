# 🚀 ETL Finance Pipeline

Pipeline de datos automatizado diseñado para la carga incremental de transacciones financieras desde sistemas legado hacia un Data Lake en AWS.

## 🛠 Tech Stack
* **Lenguaje:** Python 3.9+
* **Orquestación:** Apache Airflow
* **CI/CD:** Azure DevOps Pipelines
